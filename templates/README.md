# Template Directory

This directory contains template files for generating signs. Each template is a
`.typ` file that defines the following:
- Fonts used in the template
- Author(s) of the template
- Initial publish date
- Parameters, e.g., sign text, colours, translations, variations, etc.

All fields are required.
